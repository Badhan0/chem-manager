# CliniSync Frontend (Flutter)

This is the **Frontend (Flutter)** codebase for the **CliniSync App**.

## Key Features
- **Modern UI**: Built with a focus on ease of use.
- **Dynamic Data**: Integrates with the CliniSync backend.
- **Local Services**: Push notifications and secure data storage.
- **Profile Management**: User profile and connection handling.

## Getting Started

1. **Install Dependencies**:
   ```bash
   flutter pub get
   ```

2. **Environment Configuration**:
   Create a `.env` file in the root of the `chem_manager` directory and add:
   ```env
   BASE_URL=http://localhost:5000/api
   ```

3. **Run the App**:
   ```bash
   flutter run
   ```
